Gestión de Secciones

Consulta cupos disponibles e intercambia secciones con otros estudiantes

🔍 Consultar Secciones con Cupos Disponibles
Secciones encontradas:
ID Nombre de Sección Materia ID Cupos Disponibles
⚡ Endpoint del Backend (US-08)
GET /api/secciones/{materiaId}/disponibles

Retorna todas las secciones de la materia indicada donde cuposDisponibles > 0.

🔀 Registrar Solicitud de Intercambio
⚡ Endpoint del Backend (US-09)
POST /api/intercambios/registrar

Registra un intercambio y ejecuta el algoritmo de matching automático. Si existe un intercambio cruzado compatible, ambos pasan a estado MATCHED. De lo contrario, queda como PENDIENTE.

Body JSON de ejemplo:

{
  "estudianteId": 1,
  "materiaDeseadaId": 5,
  "materiaOfrecidaId": 3
}
